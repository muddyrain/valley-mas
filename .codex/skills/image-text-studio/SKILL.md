# 图文创作工作台

这个技能覆盖整条图文创作链路。

## 当前产品约定

- 页面由用户手动新增和删除。
- 高亮是基于选区，而不是关键词输入。
- 支持对选中的局部文字单独设置字号。
- 模板负责版式和背景，不负责自动强调某些文字。
- 编辑页预览要尽量贴近最终生成图。
- 图文卡片优先展示真实生成图，不做假的重建预览。

## 编辑规则

1. 保留每页的文本、高亮区间、局部文字样式、贴纸和模板键等信息。
2. 小心处理文本归一化，不要通过 `trim` 一类操作把字符索引挪乱。
3. 如果预览和最终导出不一致，优先统一渲染逻辑，而不是分别调两套样式。
4. 新增样式控制时，优先做成基于选区、用户容易理解的交互。
5. 图文界面要和博客界面区分开，不要简单复用博客心智。

## 校验

- 运行 `node node_modules/typescript/bin/tsc -p apps/web/tsconfig.json --noEmit`。
- 改了中文文案后运行 [$encoding-guard](/D:/my-code/valley-mas/.codex/skills/encoding-guard/SKILL.md)。
- 最终说明里写明是否涉及兼容旧数据或历史图文。

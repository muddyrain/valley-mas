# 创作空间体验

只要任务涉及创作者工作台，就使用这个技能。

## 产品意图

创作空间是管理面，不是换了皮肤的公开列表页。

## 规则

1. 创作空间卡片可以和公开卡片不一样，只要这样更利于管理。
2. 编辑、删除、查看、可见范围、发布状态都要一眼能看清。
3. 作者本人应该能从创作空间访问自己的私密内容。
4. 详情页返回应优先回到真实入口上下文。
5. 如果功能同时存在创作者入口和公开入口，改动时要保住两条路径。

## 高风险点

- 返回按钮写死回通用列表页
- 因为走公共接口而把作者私密内容判成 404
- 卡片缺少管理状态或操作入口
- 共享组件过度统一，反而削弱了工作台可用性

## 校验

至少在代码层确认一次创作空间入口到详情页的链路是否仍然完整。

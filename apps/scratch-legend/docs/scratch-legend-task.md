# 🧩 阶段 1：启动金系统（动作锁）

## 🎯 目标

👉 防止变成点击器
👉 为刮盘子提供启动资金

---

## ✅ 核心机制

### 1. 动作锁（必须）

```ts
actionDuration = 400~800ms
```

---

### 2. 收益逻辑

```ts
base = random(1, 3)
critChance = 0.05
critMultiplier = 5
```

---

### 3. 收益结算

```ts
if (isCrit) {
  gold += base * critMultiplier
} else {
  gold += base
}
```

---

### 4. 强制规则

* 动作未结束不能再次结算
* 连续点击只能增强视觉反馈
* 收益与手速无关

---

## ✅ 验收标准

* 玩家不会疯狂连点刷收益
* 1 秒 ≈ 1 次有效动作
* 30~60 秒接近刮卡成本

---

## 🚫 禁止

* 不允许直接点击获取金币
* 不允许跳阶段

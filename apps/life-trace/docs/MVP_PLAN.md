# 生活迹 Life Trace MVP 计划

## 产品定位

生活迹 Life Trace 是一个 AI 主导的个人生活计划与踪迹记录应用。它把天气、提醒、日程、打卡、图片记录和 AI 建议组合成一个生活闭环：用户先计划生活，再完成计划，并沉淀为可回看的生活踪迹。

## 第一版目标

第一版先做前端 PWA 静态 MVP，优先验证核心体验，而不是一次性接入真实天气、AI、地图和通知能力。

核心闭环：

```text
今日天气与建议 -> 创建生活计划 -> 完成计划 -> 自动生成生活踪迹 -> 在踪迹流中回看
```

## 技术方案

- 前端：React + Vite + TypeScript
- UI：Tailwind CSS + shadcn/ui dark-first 风格
- 图标：lucide-react
- 状态：Zustand + localStorage 持久化
- PWA：manifest + service worker
- 数据：第一阶段使用 mock 数据和本地状态
- 后端：后续接入现有 Valley MAS Go 服务或独立 API

## 页面范围

### 今日

- 今日天气主卡
- 未来小时天气
- AI 今日简报
- 穿衣、护肤、出门、通勤、健康、今日计划建议

### 计划

- 计划筛选：全部、今天、周末、已提醒
- 内置计划类型：电影、吃饭、运动、阅读、聚会、普通事项
- 计划卡片展示提醒状态
- 创建计划 Drawer
- 完成计划后生成踪迹

### AI

- Life AI 指挥中心
- 快捷操作：生成今日建议、创建计划、分析图片、生成踪迹、每周回顾
- 示例提示词和最近 AI 操作
- 快捷操作接入本地行为：
  - 创建计划：打开创建计划 Drawer
  - 生成今日建议：基于城市、通勤、计划数量生成本地建议
  - 生成踪迹：从最近计划生成生活踪迹
  - 每周回顾：基于计划、踪迹和打卡项生成本地总结
- 图片分析原型：
  - 支持粘贴图片 URL 或选择本地图片预览
  - 支持电影海报、美食照片、生活照片三类 mock 分析
  - 分析结果可生成计划或踪迹

### 踪迹

- 完成后的生活记录流
- 图片、标签、心情、AI 回忆文案
- 来源可以是计划、打卡或手动记录

### 我的

- 城市、工作时间、通勤方式
- 通知偏好
- 打卡项
- 数据和隐私入口
- 第一版使用 localStorage 保存偏好，并让今日页读取城市、通勤方式和工作时间。

## UI 设计规则

- 默认黑色 shadcn/ui 风格
- 背景：`#09090b`
- 卡片：`#18181b`
- 抬升面：`#27272a`
- 边框：`#27272a`
- 主文本：`#fafafa`
- 次文本：`#a1a1aa`
- 功能色：
  - 天气：`#3b82f6`
  - AI：`#06b6d4`
  - 计划：`#8b5cf6`
  - 踪迹：`#10b981`
  - 健康：`#f59e0b`
  - 风险提醒：`#f97316`

移动端约束：

- 优先适配 390px iPhone 宽度
- 页面不允许横向滚动
- 底部 Tab 统一固定
- 内容区预留底部安全距离

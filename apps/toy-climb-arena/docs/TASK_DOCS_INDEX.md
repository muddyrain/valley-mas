# Toy Climb Arena 任务文档索引

本文件用于说明 `apps/toy-climb-arena/docs` 内任务类文档的职责，避免多个 task 文档并行更新导致状态混乱。

## 当前唯一任务入口

| 文档 | 定位 | 使用方式 |
|---|---|---|
| `TASKS.md` | 当前主任务清单 | 只记录里程碑级状态、当前优先级和跨阶段任务入口。 |
| `TOY_CLIMB_PLATFORM_REMAKE_TASKS.md` | 里程碑 0.8 详细拆解 | 只记录玩具风平台、关卡重制、模型与大场景空间设计的细分任务。 |

## 非当前任务入口

| 文档 | 当前状态 | 说明 |
|---|---|---|
| `ROADMAP.md` | 历史参考 | 早期路线图，部分条目已完成或被 0.8 替代，不再作为任务状态来源。 |
| `GRAND_MAP_DESIGN.md` | 历史参考 | 旧版单线大地图方案，保留物理参数和历史设计思路，不再作为当前任务清单。 |

## 更新规则

- 新增总任务：先更新 `TASKS.md`。
- 新增 0.8 平台、模型、关卡、大场景空间任务：更新 `TOY_CLIMB_PLATFORM_REMAKE_TASKS.md`，并在 `TASKS.md` 保留一行入口。
- 不再向 `ROADMAP.md` 和 `GRAND_MAP_DESIGN.md` 写入当前状态。
- 修改任务状态时，保持 `[ ]`、`[~]`、`[x]` 三种状态语义一致。

# 玩具世界攀爬 · Game Design Document

## 游戏概述

**游戏名称：** 玩具世界攀爬（Toy Climb Arena）  
**游戏类型：** 3D 多人在线攀爬跳跃竞技  
**技术栈：** Vite + TypeScript + Three.js  
**当前阶段：** 原型 / 工程迁移阶段（v0.1）

---

## 核心玩法

- 玩家控制玩具角色，在不断向上延伸的关卡中攀爬跳跃
- 先到达顶部的玩家获胜
- 关卡由各类玩具世界障碍物构成（积木、玩具箱、弹簧板、旋转平台等）
- 支持多人同屏竞速（单机原型阶段先实现单人可玩流程）

---

## 美术主题：玩具世界

- 主色调：鲜艳饱和的玩具色系（红、黄、蓝、绿）
- 场景元素：积木块、玩具箱、弹簧跳板、乐高风格台阶、旋转风车障碍、毛绒地毯背景
- 角色风格：卡通玩具人偶（当前使用 peach/daisy GLB 模型，后续替换为玩具风格角色）
- 禁止：不直接复制任何已知游戏的模型、关卡布局或角色设计

---

## 控制方案

| 操作 | 键位 |
|---|---|
| 移动 | WASD / 方向键 |
| 跳跃 | Space |
| 奔跑 | Shift（按住） |
| 镜头 | 鼠标拖动（Pointer Lock） |
| 全屏 | F 键 |

---

## 核心循环

```
进入游戏 → 选择角色 → 加载关卡 → 攀爬竞速 → 到达终点 → 查看成绩 → 重试/下一关
```

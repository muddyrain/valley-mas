# 玩具世界攀爬 · Game Design Document

## 游戏概述

**游戏名称：** 玩具世界攀爬（Toy Climb Arena）  
**游戏类型：** 3D 单人/多人在线攀爬挑战  
**技术栈：** Vite + TypeScript + Three.js  
**当前阶段：** 原型阶段（v0.2）

---

## 核心设计方向

> **玩具世界风格的《攀爬动物》**

以《Getting Over It with Bennett Foddy》为核心体验参照——**掉落即惩罚，重新爬起来才是乐趣所在**。  
没有存档点，没有重生保护，从地面一路爬到顶，掉落回地面就从头开始。

### 设计哲学
- **掉落有重量感**：每一步都值得珍惜，掉落令人印象深刻
- **极简规则**：爬上去就赢，掉下来就重来，没有复杂机制分散注意力
- **玩具世界包装**：严酷的核心玩法套上鲜艳可爱的积木玩具外壳，形成反差萌
- **技术积累感**：玩家通过反复失败积累手感，逐渐能稳定通过曾经的难点

### 与参照游戏的区别
- 玩具积木美术风格（而非写实山石场景）
- 角色为卡通玩具人偶
- 支持多人同屏竞速（同一关卡同时在线，掉落各自独立）

---

## 核心玩法

- 玩家控制玩具角色，在积木搭建的关卡中徒手攀爬跳跃
- **无存档点**：掉落到地面后重置回起点，从头开始
- 关卡由积木、弹跳板、移动平台、不稳定平台等障碍构成
- 先到达顶部（goal）的玩家获胜（多人模式）
- 单人模式以最短用时通关为目标

---

## 美术主题：玩具世界

- 主色调：鲜艳饱和的玩具色系（红、黄、蓝、绿、橙、紫）
- 场景元素：积木块、乐高风格凸点平台、彩色棋盘格地面、积木城市背景天际线
- 角色风格：卡通玩具人偶（当前使用 peach/daisy GLB 模型，后续替换为玩具风格角色）
- 禁止：不直接复制任何已知游戏的模型、关卡布局或角色设计

|---|---|
| 移动 | WASD / 方向键 |
| 跳跃 | Space |
| 奔跑 | Shift（按住） |
| 镜头 | 鼠标拖动（Pointer Lock） |
| 全屏 | F 键 |
| 菜单 | P 键 |

---

## 核心循环

```
进入游戏 → 选择角色 → 加载关卡 → 攀爬 → 掉落回起点（重新爬）→ 登顶通关 → 查看用时 → 重试
```

---

## 掉落惩罚机制

- 玩家 Y 坐标低于地面阈值（`FALL_RESET_Y`）时触发重置
- 重置到起点平台（`startPosition`）
- 播放掉落音效（`playFall`）
- HUD 高度归零，计时继续（鼓励挑战最快用时）
- **不设存档点**，原 checkpoint 平台改为普通平台

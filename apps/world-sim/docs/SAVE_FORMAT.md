# WorldSim - 存档格式

> 版本：v0.1 · 日期：2026-05-15  
> 这份文档定义存档结构、版本和迁移规则。

## 1. 目标

- 能存、能读、能导入导出
- 新版本尽量兼容旧版本
- 存档应该尽量可重放

## 2. 顶层字段

- `saveVersion`
- `gameVersion`
- `seed`
- `createdAt`
- `updatedAt`
- `world`
- `systems`
- `settings`
- `metadata`

## 3. 存档内容

至少包括：

- 地图与地形
- 所有单位
- 所有势力
- 建筑与资源
- 时间状态
- 神力状态
- 事件摘要

## 4. 版本策略

- 每次格式变更必须提升 `saveVersion`
- 读取旧版本时先走迁移层
- 迁移失败时必须给出明确错误
- 不要默默丢字段

## 5. 确定性要求

- 存档应保存可复现所需的关键随机状态
- 影响战斗、繁殖、事件的状态要可还原
- 如果某个字段无法重放，要明确标记为非确定性数据

## 6. 导入导出

- 导出使用 JSON
- 导入前做结构校验
- 导入失败时保留当前存档

## 7. 自动存档

- 自动存档和手动存档分开
- 自动存档可以覆盖，但应保留最近一次成功版本
- 关键失败不要覆盖最后可用存档

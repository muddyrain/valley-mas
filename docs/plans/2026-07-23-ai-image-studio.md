## Task 2：Web 图片创作页面

- [x] 新增 API 类型与请求封装。
- [x] 新增 Canvas 画布，支持绘制、擦除、撤销、素材摆放与导出。
- [x] 新增模板、可见的受控提示词、比例、模型感知的目标分辨率和模型设置；尺寸能力按模型而非中转 Provider 判断，支持参考图的 Seedream 4.0 与 GPT Image 2 4K 模型优先作为默认选择。
- [x] 对生成结果执行目标像素校验；尺寸不符时失败且不转存。
- [x] 复用提示词助手，根据当前画面描述生成可预览、确认回填的图片提示词。
- [x] 新增阶段生成动效、失败态、结果历史、下载和保存资源。
- [x] 接入路由、标题和侧栏入口。

## Task 5：画布体验与参考图遵循

- [x] 修复画布卡片被设置栏拉伸、共享选中按钮 hover 丢失背景的问题。
- [x] 修复 Slider 宽度为零导致画笔粗细和素材大小无法调整。
- [x] 素材按长边等比缩放，切换画布比例时不再拉伸变形。
- [x] 参考画布作为构图真源，模板与文字不得覆盖主体数量、轮廓、姿态和空间布局。
- [x] 生成设置和历史记录明确展示参考画布是否启用、是否随任务提交。

## Task 6：历史资产独立性与回放

- [x] 保存到资源库时复制生成文件；删除资源不会再删除创作历史的图片，旧的共用对象也会被保护。
- [x] 为使用参考画布的任务保存扁平画布快照，并固化当时的模板名称与中文提示词；历史卡片可预览结果与快照，并在再次创作时还原模板、描述、模型、尺寸和画布。
- [x] 目标像素不再作为失败门槛；所有有效图片都会保存，并记录实际返回尺寸。

## 风险控制

- 内部图仅保存于 `loop.config.body`，不更改既有 Graph v4 外层数据；禁止把 React Flow 容器投影误保存为普通外层节点。
- 无限循环必须有服务端轮次与截止时间兜底；所有工具和模型调用仍受既有预算、权限与取消机制控制。
- 循环体的事件扩展保持向后兼容，历史普通运行无需迁移。
- 画布内部交互采用拖拽结束后提交与局部状态更新，避免循环体放大已有 hover/drag 渲染开销。

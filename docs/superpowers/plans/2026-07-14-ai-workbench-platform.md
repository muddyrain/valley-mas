- [x] P4：发布 API、配额和观测页面已实现并完成 JSON/SSE、未绑定 `403`、撤销失效、配额 `429` 人工验收及全量 Go/Harness 回归。
- [x] P5：公共 API 区域已提供端点、curl、JSON fetch 和 SSE fetch 示例的一键复制；API Key 支持明文创建后复制、按应用授权、加载反馈和撤销确认，已完成浏览器人工验收。
- [x] P6：工作流运行时已在建档前拒绝未开放节点，避免图触发 HTTP/代码执行或 `500`；应用版本保存知识库绑定快照和检索配置（`topK`、`minScore`、`citeSources`），调试与公共 API 读取版本快照。工作流协议已统一升级为 Graph v2：仅单开始、单结束、无环且不允许分叉；博客导入模板同步改为 v2，不再兼容 v1 图。

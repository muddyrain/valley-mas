# P13.2：知识库 2.0 实施计划

## 状态

实施中。文档可观察性与检索试验台已完成；下一步验证版本可见性，避免同时改变资料处理和运行时检索。

## 1. 文档可观察性

1. [x] 在文档 API 中明确来源类型，并新增 owner-scoped 片段预览接口。
2. [x] 预览片段按位置排序，截断内容，不返回 embedding、源文件 key 或完整原文。
3. [x] Web 文档列表展示状态；详情弹窗展示元信息与片段预览。
4. [x] 覆盖 owner 隔离、顺序与截断的服务端测试。

## 2. 检索试验台

1. [x] 复用现有默认检索阈值和向量检索，不引入第二套索引。
2. [x] 手动查询只返回命中文档、片段摘要和分数，不将全文暴露到 UI。
3. [x] 配置不可用时展示可诊断状态，不伪造结果。

## 3. 收尾

1. Web 类型检查、定向前端校验和 `cd server && go test ./...`。
2. 人工验收上传、处理中、成功、失败、详情预览和非 owner 隔离。
3. 在版本可见性验收前，不引入重排、混合检索或多模态摄取。

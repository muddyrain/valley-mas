# Valley AI 工作台平台化设计

> 状态：实施中
> 日期：2026-07-14

## 产品边界

首版面向个人创作者，所有资产按 owner 私有隔离。工作台提供智能体、工作流、知识库和受控 Valley 工具；发布后可用项目 API Key 调用。模型仅使用服务端托管 ARK，不支持团队、公开市场、用户自带 Key、任意 OpenAPI、任意 HTTP 或代码执行。

## 当前交付顺序

1. 收敛现有工作流编辑器到真实可运行节点。
2. 引入应用、版本、发布和 API Key 的统一资产模型，并兼容既有工作流和 Desktop 智能体 API。
3. 引入知识库摄取、检索和受控业务工具。
4. 提供 Web 应用列表、编辑、调试与运行观测入口。

## 安全不变量

- Graph、应用版本和工具绑定均由服务端校验；前端隐藏不是权限控制。
- 已发布版本不可原地修改，运行记录绑定版本快照。
- API Key 仅保存哈希，明文只在创建响应中出现一次。
- 文件原文、密钥和完整提示词不得写入运行公开预览或审计日志。
- 未完成安全评审的 HTTP、代码、Webhook、循环和第三方插件不开放。

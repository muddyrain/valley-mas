# 名著 CLR-1 列表与详情页结构稿

## 1. 目标

- 定版名著列表页与详情页的路由、信息层级、卡片字段与组件职责。
- 作为 CLR-2（阅读器 MVP）的前序输出：详情页是阅读器的入口。
- 与 CLD-3 元数据模型字段对齐，确保前端展示字段有对应的后端数据支撑。

---

## 2. 路由规划

| 路由 | 页面 | 说明 |
| --- | --- | --- |
| `/classics` | 名著列表页 `ClassicsList` | 支持分类筛选 + 关键词搜索 + 分页；URL 状态联动 |
| `/classic/:id` | 名著详情页 `ClassicsDetail` | 展示书目信息、版本选择、章节列表；提供进入阅读器入口 |

> 路由命名规范参考现有 `/resources` → `/resource/:id` 的单复数约定。

App.tsx 注册位置（插入 `resource/:id` 之后）：

```tsx
<Route path="classics" element={<ClassicsList />} />
<Route path="classic/:id" element={<ClassicsDetail />} />
```

RouteTitle 对应标题：

```
/classics         → 名著馆 | Valley
/classic/:id      → {书名} | Valley
```

## 3. 列表页（ClassicsList）

### 3.1 页面结构

```
┌─────────────────────────────────────────────────────┐
│  PageBanner                                          │
│  标题：名著馆  副标：经典文学作品，免费在线阅读        │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│  筛选栏（吸顶，桌面横排，移动折叠）                   │
│  [全部] [小说] [诗歌] [戏剧] [散文] [经典]           │
│  [搜索框]  ← keyword URL 联动                        │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│  书目卡片网格（桌面 4 列 / 平板 3 列 / 移动 2 列）    │
│  [ClassicBookCard] × n                              │
│  [Skeleton × 8]（加载中）                           │
│  [EmptyState]（无结果）                             │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│  分页栏（与现有 Resources 页一致）                   │
└─────────────────────────────────────────────────────┘
```

### 3.2 URL 状态

使用 `useUrlQueryState`，参数：

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `page` | number | 1 | 当前页码 |
| `category` | enum | `''` | 分类筛选，空字符串表示全部 |
| `keyword` | string | `''` | 搜索关键词，变化时重置 page |

### 3.3 ClassicBookCard 卡片字段

| 字段 | 来源（CLD-3） | 展示方式 |
| --- | --- | --- |
| `cover_url` | `Book.cover_url` | 封面图，比例 2:3，`object-cover`；无封面时展示默认书脊占位 |
| `title` / `title_zh` | `Book.title` / `Book.title_zh` | 主标题；副标题（原文书名，斜体，灰色） |
| `author.name` | `book_authors[0].author.name` | 作者名，1 行截断 |
| `category` | `Book.category` | 分类标签 chip |
| `word_count` | `Edition.word_count`（默认版本） | 字数，格式：`约 12 万字` |
| `is_translation` | `Edition.is_translation` | 若为译本，展示译者名 |

**卡片交互**：点击跳转 `/classic/:id`；hover 封面图出现「开始阅读」遮罩。

### 3.4 空态与加载态

- **加载中**：展示 8 个 `Skeleton` 卡片，与卡片尺寸一致。
- **无结果（搜索）**：`EmptyState`，说明「没有找到"${keyword}"相关名著」，提供「清除搜索」按钮。
- **无结果（分类）**：`EmptyState`，说明「该分类暂无收录作品」，提供「查看全部」按钮。

## 4. 详情页（ClassicsDetail）

### 4.1 页面结构

```
┌─────────────────────────────────────────────────────┐
│  面包屑：首页 / 名著馆 / {书名}                      │
└─────────────────────────────────────────────────────┘
┌──────────────┬──────────────────────────────────────┐
│              │  书名（大号）                          │
│  封面图       │  作者  译者（若有）                   │
│  （桌面左）   │  分类 · 时代/地域                    │
│              │  简介（展开/收起，默认 3 行）           │
│              │  字数  章节数  来源说明               │
│              │  [开始阅读]（Primary Button，大）      │
│              │  [加入书架]（Outline Button）          │
└──────────────┴──────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│  版本选择（若有多个版本）                             │
│  [朱生豪译本 ●] [梁实秋译本] [英文原版]              │
│  当前版本：来源 · 许可证 · 导入时间                  │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│  章节列表（ClassicsChapterList）                     │
│  ┌─────────────────────────────────────────────┐    │
│  │  第一章  开端          约 3,200 字  [阅读 →]  │    │
│  │  第二章  相遇          约 2,800 字  [阅读 →]  │    │
│  │  ...                                        │    │
│  └─────────────────────────────────────────────┘    │
│  桌面展示全部；移动展示前 10 章 + [展开全部]         │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│  版权声明区（折叠，默认收起）                         │
│  来源：{source_name}  链接：{source_url}             │
│  许可证：{license_type}  权利说明：{rights_note}     │
└─────────────────────────────────────────────────────┘
```

### 4.2 信息层级规则

1. **书名**：优先展示 `title_zh`（若有），否则展示 `title`；原文书名作为副标题。
2. **作者**：取 `book_authors`，按 `order` 排序；多人用 `·` 分隔。
3. **译者**：取当前选中版本的 `edition_translators`；若为原文版本则不展示译者区域。
4. **字数**：取当前版本的 `Edition.word_count`，格式化为「约 XX 万字」（< 1 万时显示实际字数）。
5. **章节数**：取 `Edition.chapter_count`。
6. **来源说明**：展示 `source_name`，不直接暴露 `source_url`（置于版权声明折叠区）。

### 4.3 版本选择交互

- 默认选中 `is_default = true` 的版本。
- 切换版本时，章节列表重新拉取对应版本的章节数据，字数和章节数同步更新。
- URL 支持 `?edition=<editionId>` 参数，刷新后保持版本选择状态。

### 4.4 章节列表字段

| 字段 | 来源（CLD-3/CLD-2） | 展示方式 |
| --- | --- | --- |
| `chapter_index` | `Chapter.chapter_index` | 章节序号（从 1 展示） |
| `chapter_title` | `Chapter.chapter_title` | 章节标题；为 null 时展示「第 N 章」 |
| `word_count` | `Chapter.word_count` | 「约 X,XXX 字」 |
| `has_footnotes` | `Chapter.has_footnotes` | 有脚注时展示「含注释」小标签 |
| 阅读按钮 | — | 点击跳转 `/classic/:id/read?chapter=<index>`（CLR-2 阅读器路由） |

### 4.5 空态与错误态

- **详情加载中**：展示 Skeleton（封面区 + 信息区 + 章节列表各有占位）。
- **书目不存在（404）**：展示「该名著不存在或已下架」，提供「返回名著馆」按钮。
- **章节列表为空**：展示「暂无章节数据，内容整理中」。

## 6. 与其他任务的接口约定

| 下游任务 | 本文输出什么 | 下游接收什么 |
| --- | --- | --- |
| CLR-2（阅读器 MVP） | 阅读器路由 `/classic/:id/read?chapter=<index>` | 阅读器从该路由接收 `id` 与 `chapter` 参数，加载对应章节 `chapter_html` |
| CLR-3（书架与最近阅读） | 「加入书架」按钮位置与状态 | 书架功能复用详情页按钮，点击后改变按钮状态为「已在书架」 |
| CLAI-1（AI 伴读入口） | 章节列表项结构（含 `chapter_index`/`word_count`） | AI 伴读入口挂载在阅读器页面，从阅读器路由获取章节上下文 |

## 7. 后端接口需求（给 server 端参考）

| 接口 | 方法 | 路径 | 说明 |
| --- | --- | --- | --- |
| 名著列表 | GET | `/public/classics` | 支持 `page`/`pageSize`/`category`/`keyword` |
| 名著详情 | GET | `/public/classics/:id` | 返回 Book + 默认 Edition + Author[] |
| 版本章节列表 | GET | `/public/classics/:id/editions/:editionId/chapters` | 返回章节概要列表（不含 `chapter_html`）|
| 单章内容 | GET | `/public/classics/:id/editions/:editionId/chapters/:index` | 返回完整 `chapter_html`/`footnotes`/`illustrations` |

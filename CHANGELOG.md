# Changelog

本文件不再作为 Valley MAS 的有效更新日志维护。

历史上这里记录过一次性修复、临时测试脚本和已下线功能，内容已经与当前代码不一致。当前仓库没有稳定发布节奏，功能变更请以 Git 历史、PR 描述、对应子项目文档和测试记录为准。

## 维护约定

- 不再向本文件追加临时流水账。
- 面向长期维护的信息应写入 `docs/PROJECT_GUIDE.md`、对应 `apps/*/README.md`、`apps/*/docs/*` 或 `server/README.md`。
- 面向一次提交的背景、约束和验证结果应写入 commit message 或 PR 描述。
- 如果以后需要正式发布日志，应重新建立按版本维护的 changelog，并只记录已发布版本。
